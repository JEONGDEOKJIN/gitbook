## 3. 배포 (npm run build, npm run start, npm run dev)

* 요약

```bash
- 빌드본 만들어서 -> .next 폴더로 이동 시키기 
npm run build 

- 빌드본 실행 
npm run start

- 다시 개발할거면, 이걸로 실행해야 해
npm run dev
```
